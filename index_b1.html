Implementing the Kalman Filter for adaptive autoregression

Introduction:

This blog post will cover implementing a Kalman Filter to perform adaptive autoregression, meaning in practise the extraction of autoregression coefficients in as continuous time as possible. If you found this page, you probably already know something about the topic, but for clarity, let's still cover the basics, and get toward implementation step-by-step.

An autoregressive (AR) process is a model for a random process, which is dependent upon it's own past values - hence the prefix 'auto'. The regression part of the word refers to process of identifying a relationship between dependent and independend variables - in our case, a value for our process X(t) based on X(t-tau) respectively. An AR model specifies the output to be dependent linearly on its own previous values and on a stochastic term (an imperfectly predictable term or 'error'). In formula notation, we write:

\begin{equation} x \left( t \right) = \sum_{i=1}^{p} x \left( t-i \right) \end{equation}
